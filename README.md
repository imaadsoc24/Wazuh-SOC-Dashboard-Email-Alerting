# Wazuh SOC Dashboard & Email Alerting

## Project Overview

Built a custom SOC monitoring dashboard in Wazuh and implemented automated email alerting for security events.

The project demonstrates real-world SOC analyst activities including security monitoring, alert triage, MITRE ATT&CK mapping, dashboard visualization, and automated alert notifications.

## Technologies Used

- Wazuh SIEM
- Wazuh Dashboard
- Docker
- Python
- Gmail SMTP
- Cron Scheduler
- Windows Event Logs
- MITRE ATT&CK Framework

## Features

- Custom SOC Monitoring Dashboard
- Security Event Visualization
- High Severity Alert Tracking
- PowerShell Alert Monitoring
- MITRE ATT&CK Mapping
- Automated Email Alerting
- Scheduled Alert Processing using Cron
- Alert Severity Filtering (Level 5 and Above)

## Dashboard Components

- Total Security Events
- High Severity Alerts
- PowerShell Alerts
- Failed Privileged Operations
- Alerts Over Time
- Top MITRE Techniques
- Top Alert Types
